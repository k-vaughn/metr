Questions for METR Workshop 2: METR Operational Structure

The second METR workshop will include focus on the following questions with the following role-based architecture presented for discussion purposes:

  • Are all relationships among METR subsystems many-to-many? (Analyze each interaction)
  • Do collectors and disseminators need to indicate their operational status and capability level?
  • How do subsystems identify and access other subsystems?
  • How do users identify what notices need to be downloaded?
  • What other issues do we need to consider?
