Questions for METR Workshop 3: Electronic Regulation Lifecycle

The third METR workshop will include focus on the following questions using the following state diagram for initial discussion purposes:

  • What are the states in the regulatory lifecycle?
  • How might regulations transfer among these states?
  • What differences might exist for different types of regulations, for examples regulations that are:
    1. Directly enacted by legislation
    2. Statically implemented based on engineering judgement
    3. Dynamically and locally implemented based on engineering judgement
    4. Part of large response plans (e.g., evacuation plans)
  • What are the responsibilities for each METR subsystem?
  • What characteristics need to be conveyed for each regulation? (e.g., location, schedule, vehicle classes affected, etc)
  • Should METR indicate when condition-based regulations are active?
  • Should evacuation plans be distributed in advance for later activation?
